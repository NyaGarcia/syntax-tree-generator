Ayuda con la gramática

Gramática

  • Para borrar la gramática, se debe pulsar el botón de la papelera.
  • Para rellenar todos los campos con una gramática de ejemplo, se debe pulsar el botón "Ejemplo".
  • Para descargar un fichero .json con la gramática introducida, se debe pulsar el botón "Descargar Gramática".

Símbolos

No terminales

  1. Haz clic en el campo bajo el título "No Terminales".
  2. Escribe el no terminal que quieras.
  3. Para añadirlo a la lista de no terminales disponibles, se debe hacer clic en el botón +, o pulsar la tecla enter.
  4. Este campo no puede estar vacío.
  5. No se permite añadir el mismo no terminal dos veces.
  6. Una vez añadido el no terminal a la lista, se mostrará debajo del campo.
  7. Para eliminar el no terminal, se debe hacer clic en la x.

Terminales

  1. Haz clic en el campo bajo el título "Terminales".
  2. Escribe el terminal que quieras.
  3. Para añadirlo a la lista de terminales disponibles, se debe hacer clic en el botón +, o pulsar la tecla enter.
  4. Este campo puede estar vacío, y se le asignará automáticamente el valor ε.
  5. No se permite añadir el mismo terminal dos veces.
  6. Una vez añadido el terminal a la lista, se mostrará debajo del campo.
  7. Para eliminar el terminal, se debe hacer clic en la x.

Reglas de Producción

No terminal

  1. Haz click en el campo con la etiqueta "Introduce un no terminal".
  2. Se abrirá una lista con los símbolos no terminales disponibles (por ejemplo: S, A, B, etc.).
  3. Selecciona uno. Solo se permite un símbolo (longitud máxima: 1).
  4. Si no hay opciones disponibles, asegúrate de haber definido los no terminales previamente.

Símbolos

  1. Haz click en el campo con la etiqueta "Introduce símbolos".
  2. Se abrirá una lista de símbolos, que incluye tanto terminales como no terminales.
  3. Se pueden seleccionar varios símbolos, que representarán el lado derecho de una regla de producción (por ejemplo: A B, a, ε, etc.).
  4. Si se quiere indicar una producción vacía, se puede dejar el campo vacío y se asignará ε automáticamente.

Añadir la regla

  • Una vez seleccionados todos los símbolos, se debe hacer clic en el botón + para agregar la regla de producción al conjunto de reglas.
  • Después de agregar la regla, los campos se limpiarán automáticamente para permitir la definición de una nueva regla.

Errores comunes

  • Dejar el campo izquierdo vacío.
  • No haber definido terminales o no terminales previamente.
  • Añadir no terminales que no se puedan derivar (carecen de regla de producción correspondiente).

Generación del árbol sintáctico

  • Para generar el árbol sintáctico de la gramática, se debe pulsar el botón "Generar Árbol"
